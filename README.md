 Expansion Cards

A simple interactive UI component where clicking a card expands it to reveal more content. Only one card can be active at a time. This project is part of Brad Traversy's "50 Projects in 50 Days - HTML, CSS & JavaScript" series.

 Technologies Used
- HTML5
- CSS3
- JavaScript

 Features
- Smooth card expansion animation
- Only one card can be active at a time
- Responsive layout for mobile and desktop
- Clean and minimal design

 Project Structure
- index.html → Main HTML file containing the card layout
- style.css → CSS file for styling, layout, and transitions
- main.js → JavaScript for handling card expansion functionality
